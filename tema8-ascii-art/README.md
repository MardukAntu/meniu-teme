\# Tema 8 – Procesor ASCII Art



Conversie imagini PNG/JPG în ASCII art + text banner ASCII.



\## Rulare local

```bash

pip install pillow

python ascii\_art.py --text "HELLO" --preview

python ascii\_art.py input.png --width 80 --chars "@#\*+=-:. " --preview

python ascii\_art.py input.png --width 80 --filter invert --output art.txt
